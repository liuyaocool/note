# windows client

## windows 发送科学wifi频段

1. v2rayN软件: 连接好节点后， 设置->参数设置->基础设置->设置端口，允许来自局域网的连接
2. sstap软件: 添加代理 -> socks5,ip,端口; 模式: 不代理中国IP
3. 打开电脑热点
4. 控制面板>网络和共享中心>更改适配器设置: 右键SSTAP 属性 共享 共享到“本地连接*10”
5. 设备连接开启的热点即可开启更新
